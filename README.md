# Implémentation d'une Solution de Stockage (NAS) - CHU Hassan II

## Description du Projet
Ce projet a été réalisé dans le cadre d'un stage technique au **CHU Hassan II de Fès**.
L'objectif était de mettre en place une solution de stockage centralisée et sécurisée.

## Missions principales
* **Installation et Configuration :** Déploiement du système de fichiers réseau avec **FreeNAS**.
* **Sécurité des données :** Mise en œuvre des listes de contrôle d'accès (**ACL**) pour restreindre les droits.
* **Gestion du stockage :** Organisation des espaces de partage pour les différents services hospitaliers.

## Environnement Technique
* Système : FreeNAS / TrueNAS.
* Protocoles : SMB / CIFS.
* Compétences : Administration système et gestion des droits d'accès.
